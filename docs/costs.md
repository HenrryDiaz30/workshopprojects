# Cost Management

Cost control mechanisms are implemented using AWS Budgets.

Budget name

workshops-budget

Budget type

Zero Spend Budget

Threshold

$0.01

The system sends email alerts if any cost is incurred.

## Estimated Costs

The architecture mainly relies on free-tier eligible services:

Lambda  
API Gateway  
DynamoDB  
S3  
CloudFront  

For educational workloads, costs are typically near $0.
